

# Informe Técnico – Página Web Personal

## 1. Introducción

El presente proyecto corresponde al desarrollo de una página web personal estática, cuyo propósito es aplicar los fundamentos del lenguaje HTML y hojas de estilo CSS en la construcción de una interfaz web estructurada.

Se prioriza el uso adecuado de etiquetas semánticas, la organización lógica del contenido y la implementación de funcionalidades básicas requeridas en el contexto académico.

---

## 2. Alcance del proyecto

La aplicación desarrollada permite presentar información personal del autor, incluyendo datos generales, intereses, formación académica y un medio de contacto mediante formulario.

El sistema no contempla persistencia de datos ni procesamiento backend, ya que se enfoca exclusivamente en el lado del cliente.

---

## 3. Estructura y organización

El documento HTML se encuentra organizado bajo un esquema semántico compuesto por:

* Encabezado principal con navegación
* Contenedor central (`main`) que agrupa las secciones informativas
* Secciones independientes para cada bloque de contenido
* Pie de página con información adicional y enlaces externos

---

## 4. Componentes principales

### 4.1 Navegación

Se implementa un menú de navegación basado en enlaces internos que permiten el desplazamiento entre secciones mediante identificadores.

### 4.2 Contenido informativo

Incluye:

* Datos personales
* Información académica
* Intereses y actividades

### 4.3 Formulario de contacto

Se incorpora un formulario HTML que permite el ingreso de:

* Nombre
* Correo electrónico
* Mensaje

Este formulario utiliza validación básica mediante atributos estándar.

---

## 5. Diseño y estilos

El diseño visual se gestiona mediante CSS, aplicando:

* Colores de fondo y tipografía
* Espaciado entre elementos
* Estilización del menú de navegación
* Personalización de enlaces (estados: hover, active, visited)
* Bordes y ajuste visual de imágenes
* Uso de imagen de fondo en el encabezado

## 9. Consideraciones finales

El desarrollo cumple con los requisitos establecidos para la actividad académica, evidenciando el uso correcto de etiquetas HTML, estructura semántica y organización del contenido.
